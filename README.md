# 喫茶店向け AI献立ツール（パスタ版）

妻が経営する喫茶店のための、**メイン食材からパスタメニューと必要な食材リストを自動提案するツール**です。

## デモ

- 公開URL: `https://あなたのユーザー名.github.io/リポジトリ名/`  
  （実際のURLに置き換えてください）

## 主な機能

- メイン食材（例: トマト・エビ・ベーコン）を1つ入力
- ランダムに3種類のパスタメニュー名を提案
- 各メニューごとに「必要な食材リスト」を表示
- スマホブラウザからも利用可能

## 使い方

1. ブラウザで上記の公開URLを開く
2. メイン食材を入力
3. 「パスタを提案」ボタンを押す
4. 提案されたメニューと食材リストを見ながら、献立を決める

# AI献立ツールを他の人にシェアする方法

このフォルダ（`index.html`・`style.css`・`script.js`）を相手に渡す方法は主に3つです。

---

## 方法1: フォルダをそのまま渡す（いちばん簡単）

**やり方**
1. 「新しいフォルダー」フォルダごと **ZIPで圧縮** する  
   （フォルダを右クリック → 「圧縮」または「送る」→「圧縮フォルダー」）
2. そのZIPを **メール・LINE・Googleドライブ・OneDrive** などで送る
3. 相手はZIPを解凍し、中にある **index.html** をダブルクリックしてブラウザで開く

**メリット** … 設定不要で誰でも開ける  
**デメリット** … 相手もファイルを受け取って開く必要がある。「リンク1つで見せる」はできない

---

## 方法2: GitHub Pages でURLを発行する（リンクで共有したいとき）

**こうなります**  
→ 無料で `https://あなたのユーザー名.github.io/リポジトリ名/` のようなURLがもらえ、そのリンクを送るだけで誰でもブラウザで開けます。

**手順の流れ**
1. [GitHub](https://github.com) でアカウントを作成（無料）
2. 新しいリポジトリを作成（名前は例: `cafe-menu-tool`）
3. このフォルダの中身（index.html, style.css, script.js など）をそのリポジトリにアップロード
4. リポジトリの **Settings** → **Pages** を開く  
   - Source で「Deploy from a branch」を選ぶ  
   - Branch で「main」、フォルダは「/ (root)」→ Save
5. 数分待つと、**Pages** のところに表示されるURL（例: `https://○○○.github.io/cafe-menu-tool/`）が使えるようになる
6. そのURLをLINEやメールで送れば、相手はクリックするだけでツールを開けます

**メリット** … リンク1つで共有できる・無料・スマホでも開ける  
**デメリット** … GitHubの操作が少し必要

---

## 方法3: Netlify Drop でURLを発行する（GitHubを使いたくないとき）

**やり方**
1. [Netlify Drop](https://app.netlify.com/drop) をブラウザで開く（Netlifyの無料アカウントでログイン）
2. 「新しいフォルダー」の中身（index.html などが入っているフォルダ）を、画面に **ドラッグ＆ドロップ**
3. しばらくすると **https://○○○.netlify.app** のようなURLが表示される
4. そのURLを共有する

**メリット** … ドラッグするだけでURLができる・無料  
**デメリット** … 更新するたびにまたドラッグしてアップロードする必要がある（GitHubと連携すると自動更新も可能）

---

## まとめ

| 目的 | おすすめ |
|------|----------|
| とにかく簡単に渡したい | **方法1**（ZIPで送る） |
| 「このリンク開いて」と送りたい | **方法2**（GitHub Pages）または **方法3**（Netlify Drop） |

喫茶店のスタッフや知人に「使ってみて」と渡すだけなら **方法1**、SNSやメッセージでリンクを貼って共有したいなら **方法2** か **方法3** が向いています。

# GitHub Pages での公開手順

## 手順

1. GitHub リポジトリの Settings → Pages を開く
2. Build and deployment
   - Source: `Deploy from a branch`
   - Branch: `main` / Folder: `/ (root)`
3. Save を押す
4. 数分待つと、`Your site is live at ...` に公開URLが表示される

## 更新方法

- `index.html` / `style.css` / `script.js` を変更して main ブランチに反映すると、自動で公開サイトも更新される
- 反映には数十秒〜数分かかることがある

# 機能アイデアメモ

## すぐに試したいアイデア

- コピーボタン：メニュー名＋食材リストをワンクリックでコピー
- 印刷用スタイル（@media print）で献立表として出力
- メイン食材の候補をプルダウンで選べるようにする

## もう少し先のアイデア

- ドリンク・スイーツ版の画面を追加
- 「さっぱり」「こってり」「短時間で作りたい」など条件指定
- AI（LLM）を使ってメニュー名や説明文を自動生成
- 在庫・原価を入力して、それを考慮した提案

## メモ

- 良さそうなアイデアが出たら、とりあえずここに書き出しておく
- 実装することが決まったら、GitHub Issue を作成してタスク化する

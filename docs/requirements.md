# 要件・利用シナリオ（喫茶店向け AI献立ツール）

## 想定ユーザー

- 妻（喫茶店オーナー）
- 必要に応じてスタッフも利用

## 想定シーン

- 毎月の「今月のパスタ」候補を素早く決めたい
- 仕入れた食材をできるだけ無駄なく使いたい
- メニュー名のアイデア出しを楽にしたい

## 必須要件（現状）

- メイン食材を1つ入力すると、3つのパスタメニュー名が出る
- 各メニューごとに必要な食材リストが表示される
- スマホのブラウザからもストレスなく利用できる
- インストール不要（URLを開くだけ）

## 将来追加したい要件（候補）

- ドリンク・スイーツの提案
- １か月分の献立表の自動生成
- 在庫・原価を入力して、それを考慮した提案
- 印刷・PDF出力用のレイアウト
